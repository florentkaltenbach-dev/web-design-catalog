Accessibility Features

  • Keyboard navigation: Use Tab to navigate, Enter/Space to select
  • Screen reader support: Course cards and form fields are properly labeled
  • Focus indicators: Clear visual focus states on all interactive elements
  • Color contrast: Meets WCAG AA standards (4.5:1 for text)
  • Reduced motion: Respects prefers-reduced-motion preference
  • Error handling: Real-time validation with clear error messages

About This Pattern

The Lightning Booking pattern is optimized for busy professionals who know exactly what they want. Every interaction is designed to minimize friction and maximize speed.

Key Design Decisions:

  • Single-page flow: No modals or multi-step wizards. Everything happens inline.
  • Instant feedback: Course selection shows immediate visual response.
  • Minimal form fields: Only name, email, and optional phone. No unnecessary data collection.
  • Smart defaults: Form auto-focuses to speed up data entry.
  • Immediate confirmation: Success state appears instantly with actionable next steps.

Target Completion Time:

15 seconds from selection to confirmation
