Web Design Pattern Catalog

15 production-ready design patterns for calm, trust-building web experiences

Design Philosophy

This catalog contains adaptive design patterns for trust-building web experiences. Each pattern follows core principles: calm aesthetics, trust-building interactions, warm language, and WCAG AA accessibility.

🛡

Trust First

Build credibility before asking for action

♥

Non-Clinical

Use warm, empowering language

⚡

Accessible

WCAG AA compliant, keyboard navigable
